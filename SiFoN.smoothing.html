Plots smooth and raw sequence class scores side by side to compare visualizations.

Parameters

  • df (Pandas DataFrame) – Dataframe with the columns “Raw Scores”, “Smoothed Scores”, and “Position”.

  • figname (string) – Name of file that figure will be saved as.

  • sequence_class_name (string) – The name of the sequence class to apply rolling average to.

  • s (float, optional) – Size of data points, default is 0.2.

Plots smooth and raw sequence class scores side by side to compare visualizations.

Parameters

  • df (Pandas DataFrame) – Dataframe with the columns “Raw Scores”, “Smoothed Scores”, and “Position”.

  • window_length (int, optional) – The length of the Savitzy-Golay filter window, default is 801.

  • polyorder (int, optional) – The order of the polynomial used to fit the samples in Savitzy-Golay filter, default is 4.

  • sm (Boolean) – Whether or not to apply Softmax, default is True.

SiFoN.smoothing.smooth_class(df, sequence_class_name, window=15)[source]¶

Applies a rolling average on sequence claas scores for a particular class.

Parameters

  • df (Pandas DataFrame) – Dataframe with the columns “Raw Scores”, “Sequence Class”, and “Position”. This function is designed for SNPs that are all adjacent to one another. “Raw scores” are Sei sequence class scores and “Sequence Class” denotes the corresponding sequence class of the score. It is assumed that all SNPs are along the same chromosome, allowing for smoothing via adjacency. “Position” denotes the position of the SNP on the genome.

  • sequence_class_name (string) – The name of the sequence class to apply rolling average to.

  • window (int, optional) – Window of rolling average, default is 15.

Returns

Dataframe containing scores for the sequence_class_name class after the application of a rolling average. Note that elements on the window edge are removed from the dataframe such that the output will have a smaller size than the input dataframe.

Return type

Pandas DataFrame
